## 按钮 

:::demo 使用`type`给Button添加固定类型
```html
<f-row>
<f-button>默认按钮</f-button> 
<f-button type="primary">主要按钮</f-button> 
<f-button type="info">主要按钮</f-button> 
<f-button type="warn">警告按钮</f-button> 
<f-button type="error">错误按钮</f-button> 
<f-button type="success">成功按钮</f-button>
</f-row>

<f-row>
<f-button>朴素按钮</f-button> 
<f-button type="primary" plain>主要按钮</f-button> 
<f-button type="info" plain>主要按钮</f-button> 
<f-button type="warn" plain>警告按钮</f-button> 
<f-button type="error" plain>错误按钮</f-button> 
<f-button type="success" plain>成功按钮</f-button>
</f-row>
 

```
:::
